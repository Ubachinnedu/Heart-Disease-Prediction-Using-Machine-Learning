##  Overview

This project uses machine learning techniques to predict the likelihood of heart disease based on patient health data. The objective is to identify individuals at higher risk and demonstrate how predictive analytics can support healthcare decision-making.

## Objectives

- Analyze patient health records
- Identify key factors associated with heart disease
- Build and evaluate a predictive machine learning model
- Support early detection through data-driven insights

##  Tools & Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Project Workflow

- Data collection and exploration
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training
- Model evaluation and interpretation

##  Machine Learning

The project applies classification algorithms to predict whether a patient is likely to have heart disease based on clinical attributes.

## Results

The model successfully identified important risk factors and demonstrated the potential of machine learning in supporting healthcare professionals with early diagnosis.
